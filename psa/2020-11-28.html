Assync and Interstudent Communication are back!

You might not know them because they have been broken for a while:

  • Assync allows you to sync your assignments between devices. You can find it in Options.
  • Interstudent Communication is a public chat where vowels are banned for moderation reasons. It resides in the Utilities tab.

The time left in the period now also shows in the favicon, so it'll still show if you pin UGWA. Because of this, UGWA now has a new logo.

Other changes:

  • Fixed an old bug where UGWA wouldn't update the tab title while viewing a different day's schedule.
  • Improved UGWA's start up time.
  • UGWA is now less vulnerable to XSS.
